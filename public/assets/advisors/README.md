# Advisor Photos

This folder contains photos of ETH Blockchain Club advisors.

## Guidelines for adding photos:

**Format**: Use JPG or PNG format
**Size**: Recommended size is at least 400x400px (square)
*Quality**: Use professional or high-quality photos with good lighting
**Background**: Preferably neutral or consistent backgrounds
